# Carbon Capture Trading Cluster (ABM)

This project implements an agent-based model (ABM) to simulate industrial adoption of carbon capture technology under a dynamic carbon tax.

## Features
- Agent-based simulation of 100 industrial plants
- Carbon tax policy agent
- Avalanche detection and SOC behavior
- Connectivity sweep analysis

## Results
- Power-law avalanche distribution
- Phase transition to fully green cluster
